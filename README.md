# 🎓 Kerugoya Boys High School Website

A modern, responsive high school website built using **HTML, CSS, and JavaScript**.  
This project showcases a professional school web presence with a focus on **user experience, accessibility, and mobile responsiveness**.

---

## 🌐 Live Demo
👉 (https://kerugoyaboys.netlify.app/)

---

## 📌 Features

- 🏫 Homepage with hero section
- 📚 Academics section
- 📝 Admissions page
- 🎓 Student Life section
- 🖼 Gallery page
- 📱 Fully responsive design (mobile + desktop)
- 🍔 Mobile navigation (hamburger menu)
- 🎨 Modern UI/UX design

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Tailwind + Custom CSS)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts
